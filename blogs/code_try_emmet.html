尝试使用Emmet语法

尝试了一下Emmet的语法，发现还是很强大的。所以决定使用Emmet写这篇博客，记录下常用用法，以后也要坚持使用。

html代码缩写:

  • 初始化:!或html:5
  • 添加类,id,文本,属性:.item #item {text} [href="http://image.so.com"]
  • 嵌套:> + ^
  • 分组:(.header>h1)+(.content>p)+(.footer>p)
  • 隐式标签:.item
  • 多个元素:ul>.item*5

CSS缩写:

  • 常用属性:w200+h100+m20
  • 供应商兼容代码:trs
  • 渐变:lg(left,#fff 50%,#000)

  • Lorem:lorem
